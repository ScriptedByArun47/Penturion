PENTURION: Automated Pentesting Assistant



🚀 Overview
PENTURION is a smart, rule-based tool designed to automate network penetration testing. It uses an inference engine to scan targets, gather info, and find vulnerabilities. It's built to handle the initial stages of a pentest, freeing up security pros for deeper work.



✨ Features

Rule-Based Scanning: Runs Nmap commands and other actions based on dynamic rules.

Intelligent Fact Management: Stores discovered data (like open ports and services) to guide future scans.

Progressive Scan Stages: Moves through stages like initial discovery and detailed vulnerability checks.

Deep Service Enumeration: Automatically dives into services like SSH, SMTP, MySQL, FTP, and SMB.

Web Application Checks: Performs detailed scans for web servers (WebDAV, login pages, directory brute-forcing).

General Vulnerability Checks: Looks for common network weaknesses (DNS, SNMP).

Exploit Suggestions: Offers actionable advice for exploiting found vulnerabilities.
